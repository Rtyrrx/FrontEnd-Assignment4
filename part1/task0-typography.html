Responsive Typography (Task 0)

This page demonstrates changing font sizes using CSS media queries for mobile, tablet, and desktop breakpoints.

Why responsive type?

Readable typography improves UX across devices. On larger screens, we can comfortably increase type size and spacing. On smaller screens, we keep sizes compact to fit content and maintain legibility.

Breakpoints used

Tablet at 576px and Desktop at 992px. These match common Bootstrap breakpoints but are applied here using plain CSS only.
